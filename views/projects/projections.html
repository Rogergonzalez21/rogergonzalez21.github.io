About the Project

Projections.io is an iOS app that allows the user to control content directly on their TVs, capture clips, edit them and share them on his social networks.

What I did
  • Development lead.
  • Backend development.
  • Developed an API Scrapper to load all the movies/shows to a database.
  • Firebase implementation (Auth + Database).
  • Setup and deployment on AWS.
